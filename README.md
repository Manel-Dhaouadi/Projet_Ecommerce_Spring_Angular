# Projet E-commerce Full Stack

## Description
Application web de vente en ligne développée avec Spring Boot et Angular.
Ce projet a été réalisé dans le cadre de ma formation en développement full stack.

## Technologies utilisées
- **Backend** : Spring Boot, MongoDB, JWT, Redis
- **Frontend** : Angular, TypeScript, Bootstrap
- **Outils** : Maven, Git, Postman

## Fonctionnalités
- Authentification JWT
- Gestion de catalogue produits
- Panier d'achat
- Recherche de produits
- Gestion des commandes

## Installation
1. Installer MongoDB, Redis (optionnel)
2. Cloner le repository
3. Lancer le backend : `mvn spring-boot:run`
4. Lancer le frontend : `npm install && ng serve`
